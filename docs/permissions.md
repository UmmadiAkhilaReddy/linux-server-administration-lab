##Permission

```bash
chown username filename #used to change owner of file
chgrp groupname file #used to change group of file
chown username:groupname filenames #used to change group and user at a time
chown username:groupname foldername #used to change group and user at a time
chown -R username:groupname folder #change grp and user to folder and inside folder at a time
chown username:groupname folder/* #change user and grp inside the folder
```
